# ATM Machine Simulator üè¶

This is a simple Python program that simulates basic ATM operations.
It is designed for beginners to understand core Python concepts.

## Features
- User ID and PIN verification
- Check account balance
- Deposit money
- Withdraw money
- Exit option

## Concepts Used
- Dictionary
- Loops
- Conditional statements
- Functions (optional)

## How to Run
1. Open the file `atm.py`
2. Run the program using:
   python atm.py

## Sample Output
- Displays ATM menu
- Performs selected banking operations
